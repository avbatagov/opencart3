# opencart3
модули для opencart3

#### Произвольные блоки преимуществ, отзывов - [https://github.com/avbatagov/opencart3/tree/main/random_blocks](https://github.com/avbatagov/opencart3/tree/main/random_blocks) 
Простые модули для добавления блоков преимуществ, отзывов. Реализовано через стандартную модель "setting/module". Добавляются в нужное место страницы через макеты.

Дополнительно создан функционал возможности добавления таких блоков на страницы разделов каталога (в админке у каждой сущности блока можно указать в каких категориях необходимо его показывать).
Для того чтобы активировать этот функционал необходимо будет внести правки в файл модели /catalog/model/setting/module.php:13-23 (модифицированный файл есть в репозитории)
После чего добавить нужную логику в контроллер категории /catalog/controller/product/category.php:101-134 (модифицированный файл есть в репозитории)
А также вывести их в шаблоне категории TWIG /www/ocstore.alexey-batagov.ru/catalog/view/theme/default/template/product/category.twig:170-275
